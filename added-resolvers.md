# Added Resolvers

- `isTopFloorHeated` (boolean, topFloor input, defaults to `true`)
- `topFloorArea` (number, required input)
- `topFloorType` (string, topFloor input, config year-band fallback via `buildingYear`)
- `topFloorInnerSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.UPWARD`)
- `topFloorOuterSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.UPWARD`)
- `topFloorThermalConductivity` (number, config `topFloor.thermalConductivity`)
- `topFloorInsulationThickness` (number, input override or config `assumedInsulationThickness`)
- `topFloorInsulationResistance` (Rsi + d/λ + Rse)
- `topFloorHasInsulation` (boolean, input override, defaults to `false`)
- `roofHasInsulation` (boolean, input override, defaults to `false`)
- `betweenRafterRoofResistance` (insulationResistance / (reductionFactor + 1))
- `roofConstructionUValue` (number, keyed by `roofConstructionType` + year-band via `roofYear`)
- `roofConstructionResistance` (1 / roofConstructionUValue)
- `roofInsulationResistance` (Rsi + d/λ + Rse)
- `topFloorConstructionUValue` (number, keyed by `topFloorType` + year-band via `buildingYear`)
- `topFloorConstructionResistance` (1 / topFloorConstructionUValue)
- `topFloorHeatLossFactor` (number, config `topFloor.heatLossFactor`)
- `topFloorHeatLoss` (area * uValue * heatLossFactor)
- `topFloorThermalResistance` (constructionResistance + insulationResistance if hasInsulation, else constructionResistance)
- `topFloorUValue` (1 / topFloorThermalResistance)
- `outerWallHasInsulation` (boolean, input override, defaults to `false`)
- `outerWallConstructionType` (string, input override or config year-band fallback via `buildingYear`)
- `outerWallConstructionUValue` (number, keyed by `outerWallConstructionType` + year-band via `buildingYear`)
- `outerWallArea` (number, required input)
- `outerWallConstructionResistance` (1 / outerWallConstructionUValue)
- `outerWallThermalResistance` (constructionResistance + insulationResistance)
- `outerWallUValue` (1 / outerWallThermalResistance)
- `outerWallHeatLossFactor` (number, config `outerWall.heatLossFactor`)
- `outerWallHeatLoss` (area × uValue × heatLossFactor) → added to heatLossSum
- `baseSlabIsHeated` (boolean, input override, defaults to `false`) — registered but unused
- `baseSlabArea` (number, required input)
- `baseSlabConstructionType` (string, input override or config year-band fallback via `buildingYear`)
- `baseSlabConstructionUValue` (number, keyed by `baseSlabConstructionType` + year-band via `buildingYear`)
- `baseSlabConstructionResistance` (1 / baseSlabConstructionUValue)
- `baseSlabHasInsulation` (boolean, input override, defaults to `false`)
- `baseSlabInsulationThickness` (number, input override or config `assumedInsulationThickness`)
- `baseSlabInnerSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.DOWNWARD`)
- `baseSlabOuterSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.DOWNWARD`)
- `baseSlabThermalConductivity` (number, config `baseSlab.thermalConductivity`)
- `baseSlabInsulationResistance` (Rsi + d/λ + Rse, 0 if !hasInsulation)
- `baseSlabThermalResistance` (constructionResistance + insulationResistance)
- `baseSlabUValue` (1 / baseSlabThermalResistance)
- `baseSlabHeatLossFactor` (number, config `baseSlab.heatLossFactor`)
- `baseSlabHeatLoss` (area × uValue × heatLossFactor) → added to heatLossSum
- `outerWallInnerSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.HORIZONTAL`)
- `outerWallOuterSurfaceThermalResistance` (number, config heat keyed by `HeatFlowDirection.HORIZONTAL`)
- `outerWallThermalConductivity` (number, config `outerWall.thermalConductivity`)
- `outerWallInsulationThickness` (number, input override or config `assumedInsulationThickness`)
- `outerWallInsulationResistance` (Rsi + d/λ + Rse, 0 if !hasInsulation)
